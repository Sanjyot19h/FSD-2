## Designing UI using component libraries (Material UI / Bootstrap)
 
 ## Technologies Used
- **React (Vite)**
- **Material UI**
- **React Bootstrap**
- **JavaScript (JSX)**
- **CSS**

Multiple UI components such as Button, TextField, Select, Checkbox, and Rating were implemented using component-based architecture.  
These components demonstrate how modern UI libraries help in building clean, reusable, and responsive interfaces efficiently.
